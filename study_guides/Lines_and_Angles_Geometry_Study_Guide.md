## Chapter 1: Basic Angle Relationships

### 1.1 Angle Types and Definitions

| Angle Type | Definition | Measure Range |
|------------|------------|---------------|
| **Acute** | Less than 90° | 0° < θ < 90° |
| **Right** | Exactly 90° | θ = 90° |
| **Obtuse** | Greater than 90°, less than 180° | 90° < θ < 180° |
| **Straight** | Exactly 180° | θ = 180° |
| **Reflex** | Greater than 180°, less than 360° | 180° < θ < 360° |

### 1.2 Angle Pairs

**Complementary Angles**: Two angles that sum to 90°
- If ∠A + ∠B = 90°, then ∠A and ∠B are complementary
- SAT Tip: If one angle is x, the other is 90° - x

**Supplementary Angles**: Two angles that sum to 180°
- If ∠A + ∠B = 180°, then ∠A and ∠B are supplementary
- SAT Tip: If one angle is x, the other is 180° - x
- Linear pairs are always supplementary

**Vertical Angles**: Opposite angles formed by intersecting lines
- **Theorem**: Vertical angles are congruent (equal in measure)
- SAT Trap: Adjacent angles are supplementary, vertical angles are equal

**Adjacent Angles**: Share a common vertex and side, no overlap
- Linear pair = adjacent + supplementary

## Chapter 2: Parallel Lines and Transversals

### 2.1 Transversal Definitions

A **transversal** is a line that intersects two or more other lines at distinct points.

When a transversal crosses parallel lines, it creates **8 angles** with specific relationships:

```
       Line 1 (parallel)
    a  /  b
     /______\
c   /  d     \  e   (Transversal)
   /__________\
f  /  g       \  h   Line 2 (parallel)
   /____________\
```

### 2.2 Angle Relationships with Parallel Lines

| Pair Type | Angles | Relationship | SAT Keywords |
|-----------|--------|--------------|--------------|
| **Corresponding** | a&e, b&f, c&g, d&h | **Congruent** | "corresponding angles" |
| **Alternate Interior** | c&f, d&e | **Congruent** | "alternate interior" |
| **Alternate Exterior** | a&h, b&g | **Congruent** | "alternate exterior" |
| **Consecutive Interior** (Same-Side Interior) | c&e, d&f | **Supplementary** (sum = 180°) | "same-side interior", "consecutive interior" |
| **Vertical** | a&d, b&c, e&h, f&g | **Congruent** | "vertical angles" |
| **Linear Pairs** | a&b, c&d, e&f, g&h | **Supplementary** | "linear pair", "adjacent" |

### 2.3 The "Z", "F", "C" Memory Patterns

- **Z-Pattern (Alternate Interior)**: Look for a Z shape → angles are equal
- **F-Pattern (Corresponding)**: Look for an F shape → angles are equal  
- **C-Pattern (Consecutive Interior)**: Look for a C shape → angles sum to 180°

### 2.4 Proving Lines Parallel (Converse Theorems)

If any of these are true, the lines are parallel:
1. Corresponding angles are congruent
2. Alternate interior angles are congruent
3. Alternate exterior angles are congruent
4. Consecutive interior angles are supplementary

## Chapter 3: SAT Problem Types and Strategies

### 3.1 Type 1: Find the Missing Angle (Direct Application)

**Problem**: In the figure, line l || line m. If ∠1 = 65°, find ∠5.

**Solution**:
- ∠1 and ∠5 are corresponding angles
- Corresponding angles are congruent when lines are parallel
- ∠5 = 65°

**Practice**:
1. Lines p || q, transversal t. If one alternate interior angle = 42°, what is the other alternate interior angle?
2. Lines are parallel. One consecutive interior angle = 110°. Find the other.
3. ∠A and ∠B are vertical angles. ∠A = 3x + 10, ∠B = 5x - 30. Find x and the angle measure.

### 3.2 Type 2: Algebraic Angle Problems

**Problem**: In the diagram, line a || line b. ∠1 = (3x + 15)°, ∠2 = (5x - 25)°. ∠1 and ∠2 are alternate interior angles. Find x and the angle measures.

**Solution**:
- Alternate interior angles are congruent
- 3x + 15 = 5x - 25
- 40 = 2x
- x = 20
- ∠1 = 3(20) + 15 = 75°
- ∠2 = 5(20) - 25 = 75° ✓

**SAT Trap**: Students often set them supplementary instead of congruent. Check the angle pair type!

**Practice**:
1. Corresponding angles: (2x + 30)° and (4x - 10)°. Find x.
2. Consecutive interior: (x + 40)° and (3x - 20)°. Find x.
3. Vertical angles: (5x - 10)° and (3x + 30)°. Find x.

### 3.3 Type 3: Multi-Step Angle Chasing

**Problem**: Lines l, m, n are parallel. Transversals t1 and t2 create the angles shown. Find x.

```
t1:  ∠A = 50°
      \
       \ l (parallel)
        \
         \ m (parallel)
          \
           \ n (parallel)
            \
t2:         ∠B = x°
```

**Strategy**: 
1. Use corresponding/alternate interior to "carry" known angles across parallel lines
2. Use triangle sum (180°) or linear pairs (180°) when needed
3. Work step by step, labeling each angle you find

**Practice**: Given parallel lines and multiple transversals, find the value of x when angles are expressed as algebraic expressions.

### 3.4 Type 4: Angles in Polygons with Parallel Lines

**Key Theorems**:
- **Triangle Sum**: Interior angles of a triangle = 180°
- **Exterior Angle Theorem**: Exterior angle = sum of two remote interior angles
- **Polygon Interior Sum**: (n-2) × 180° for n-sided polygon

**SAT Pattern**: A triangle formed by transversals cutting parallel lines. Use parallel line theorems + triangle sum.

**Example**: 
```
Line l || line m
Transversal t1 creates ∠1 = 60° on line l
Transversal t2 creates ∠2 = 70° on line m
The transversals intersect between the lines forming a triangle.
Find the third angle of the triangle.
```

**Solution**:
- Angle on line l corresponding to ∠1 = 60° (on the triangle)
- Angle on line m corresponding to ∠2 = 70° (on the triangle)
- Third angle = 180° - 60° - 70° = 50°

## Chapter 5: Common SAT Traps and How to Avoid Them

| Trap | Description | Prevention |
|------|-------------|------------|
| **Wrong Angle Pair** | Treating consecutive interior as congruent | Label angle pairs before solving |
| **Algebraic Errors** | Sign errors when solving for x | Check: plug x back into both expressions |
| **Assuming Parallel** | Using parallel theorems when lines aren't marked parallel | Only use theorems if lines are stated/marked parallel |
| **Exterior vs Interior** | Confusing alternate exterior with alternate interior | Draw the "Z" and "F" patterns |
| **Angle vs Angle Measure** | Answering x when question asks for angle measure | Always reread: "Find x" vs "Find the angle" |
| **Supplementary vs Complementary** | Using 90° instead of 180° or vice versa | "Straight line" = 180°, "Right angle" = 90° |

## Chapter 6: Practice Problems with Solutions

### Problem 1 (Easy)
Lines a and b are parallel. A transversal intersects them forming angles 1-8. If m∠3 = 55°, find m∠6.

**Solution**: ∠3 and ∠6 are alternate interior angles. They are congruent.
**Answer**: 55°

---

### Problem 2 (Medium)
In the figure, line p || line q. The measure of ∠A is (2x + 10)° and the measure of its corresponding angle ∠B is (4x - 30)°. Find the measure of ∠A.

**Solution**: Corresponding angles are congruent.
2x + 10 = 4x - 30
40 = 2x
x = 20
∠A = 2(20) + 10 = 50°
**Answer**: 50°

---

### Problem 3 (Medium)
Lines l and m are parallel. A transversal creates consecutive interior angles measuring (3x + 15)° and (5x - 25)°. Find x.

**Solution**: Consecutive interior angles are supplementary.
(3x + 15) + (5x - 25) = 180
8x - 10 = 180
8x = 190
x = 23.75
**Answer**: x = 23.75

---

### Problem 4 (Hard)
In the diagram, AB || CD || EF. Line GH intersects them at points J, K, L respectively. If ∠GJA = 110° and ∠EKL = 70°, what is the measure of ∠JKL?

**Solution**:
- ∠GJA and ∠JKC are corresponding (since AB || CD) → ∠JKC = 110°
- ∠JKC and ∠JKL are a linear pair → ∠JKL = 180° - 110° = 70°
- Alternatively: ∠EKL = 70° and ∠JKL are alternate interior (CD || EF) → ∠JKL = 70°
**Answer**: 70°

---

### Problem 5 (Hard - Triangle + Parallel Lines)
Line m || line n. Transversals t1 and t2 intersect m at points A and B, and n at points C and D. The transversals intersect at point E between the lines. In triangle ABE, ∠A = 40° and ∠B = 60°. Find ∠CED.

**Solution**:
- In ΔABE: ∠E = 180° - 40° - 60° = 80°
- ∠AEB and ∠CED are vertical angles → ∠CED = 80°
- Check: Corresponding angles from parallel lines confirm this
**Answer**: 80°

---

### Problem 6 (SAT Style - Algebraic)
In the figure, line p || line q. Ray r bisects ∠1. If m∠1 = (8x - 20)° and m∠2 = (4x + 10)°, where ∠1 and ∠2 are consecutive interior angles, find the measure of each half-angle created by the bisector.

**Solution**:
- Consecutive interior angles are supplementary: (8x - 20) + (4x + 10) = 180
- 12x - 10 = 180 → 12x = 190 → x = 190/12 = 95/6 ≈ 15.83
- ∠1 = 8(95/6) - 20 = 760/6 - 120/6 = 640/6 = 320/3 ≈ 106.67°
- Bisector creates two angles of (320/3)/2 = 160/3 ≈ 53.33° each
**Answer**: 160/3° or 53⅓°

---

### Problem 7 (Polygon + Parallel Lines)
A regular pentagon has one side extended to form an exterior angle. A line parallel to the adjacent side is drawn through the vertex. Find the acute angle between the parallel line and the extended side.

**Solution**:
- Interior angle of regular pentagon = (5-2)×180/5 = 108°
- Exterior angle = 180 - 108 = 72°
- The parallel line creates corresponding angles with the transversal
- The acute angle = exterior angle = 72°
**Answer**: 72°

---

### Problem 8 (Multiple Transversals)
Three parallel lines are cut by two transversals. The transversals intersect between the first and second parallel lines. Given angles: 50°, 60° at the intersection on the first line. Find the angle marked x at the intersection on the third line.

**Solution**:
- Use corresponding angles to transfer the 50° and 60° to the intersection on the second line
- The triangle formed has angles 50°, 60°, and 70° (sum = 180°)
- The 70° angle corresponds to angle x on the third line
**Answer**: 70°

## Chapter 7: Decision Tree for SAT Lines/Angles Problems

```
START: Read problem, identify all given angles and lines
   │
   ├─ Are lines marked parallel? → YES: Use parallel line theorems
   │                              NO: Use only vertical angles, linear pairs, triangle sum
   │
   ├─ What angle pairs are involved?
   │     ├─ Corresponding / Alt Interior / Alt Exterior / Vertical → CONGRUENT
   │     ├─ Consecutive Interior / Linear Pair → SUPPLEMENTARY (180°)
   │     └─ Complementary → SUM = 90° (rare, usually right angle context)
   │
   ├─ Are there algebraic expressions?
   │     ├─ Set up equation based on angle relationship
   │     ├─ Solve for variable
   │     ├─ SUBSTITUTE BACK to find angle measures
   │     └─ CHECK: Do angles make sense? (0° < angle < 180° typically)
   │
   ├─ Multiple transversals / triangles?
   │     ├─ "Carry" angles across parallel lines using corresponding angles
   │     ├─ Use triangle sum (180°) or exterior angle theorem
   │     └─ Look for vertical angles at intersections
   │
   └─ FINAL: Reread question - what exactly is asked? (x? angle measure? sum?)
```

## Chapter 9: Advanced Topics (For 750+ Scores)

### 9.1 Angle Bisectors with Parallel Lines
If a ray bisects an angle formed by a transversal and a parallel line, it creates equal angles that maintain parallel relationships.

### 9.2 Perpendicular Transversals
If a transversal is perpendicular to one parallel line, it's perpendicular to ALL parallel lines. All angles = 90°.

### 9.3 Parallel Line Proportionality (Coordinate Geometry Connection)
If three parallel lines cut two transversals, the segments are proportional:
```
AB/BC = DE/EF  (where A,B,C on one transversal, D,E,F on other)
```

### 9.4 Angle Preservation Under Similarity
Parallel lines preserve angle measures in similar figures. Used in coordinate geometry proofs.

**Type 4: Composite Solids** - Add/subtract volumes of simple shapes
**Type 5: Inscribed/Circumscribed** - Sphere in cube, cylinder in sphere, etc.

- **Central Claim with Data**: "Study shows 60% reduction" → treatment group had 60% fewer events than control
- **Correlation vs. Causation Language**: "Associated with" ≠ "caused"; "linked to" ≠ "resulted from"

- **Sample Size Claims**: "Based on 500 participants" → assess reliability (larger = more reliable)

1. **Identify the specific claim** being referenced

#### Advanced Angle Theorems
| Theorem | Statement | Application |
|---------|-----------|-------------|
| **Exterior Angle Theorem** | Exterior angle = sum of two remote interior angles | Find missing angles fast |

#### Special Triangle Centers (Coordinate Geometry)
| Center | Construction | Coordinates (if vertices known) | Key Property |
|--------|--------------|----------------------------------|--------------|
| **Centroid** | Medians intersect | $(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3})$ | Divides medians 2:1 |

| **Orthocenter** | Altitudes intersect | - | Acute: inside; Right: at vertex; Obtuse: outside |

#### Euler Line
- Centroid, circumcenter, orthocenter are **collinear**
- Centroid is 2/3 of the way from orthocenter to circumcenter
- **Only equilateral triangles** have all centers coincide

### 🔴 NEW: Quadratics & Systems Advanced (from Math_H2)
*Extracted from 2026 Summer SAT/ACT @ AHA Math_H2pdf (July 9, 2026)*

#### Quadratic Forms & Strategic Rewriting
| Form | Equation | Best For |
|------|----------|----------|
| **Standard** | $ax^2 + bx + c$ | Factoring, quadratic formula, $c$ = y-intercept |
| **Vertex** | $a(x-h)^2 + k$ | Vertex $(h,k)$, max/min, transformations |
| **Factored** | $a(x-r_1)(x-r_2)$ | Roots/zeros, x-intercepts, sign analysis |

#### Converting Between Forms
- **Standard → Vertex**: Complete the square
  - $x^2 + 6x + 2 = (x^2 + 6x + 9) - 9 + 2 = (x+3)^2 - 7$
- **Standard → Factored**: Factor or use quadratic formula to find roots
- **Vertex → Standard**: Expand $(x-h)^2$

#### Systems: Linear-Quadratic & Quadratic-Quadratic
**Strategy**: Substitute linear into quadratic → solve → back-substitute
- **0 solutions**: Line misses parabola (discriminant < 0)
- **1 solution**: Line tangent (discriminant = 0)
- **2 solutions**: Line intersects twice (discriminant > 0)

**Example**: $y = x+1$ and $y = x^2 - 2x + 3$
- $x+1 = x^2 - 2x + 3 \rightarrow x^2 - 3x + 2 = 0 \rightarrow (x-1)(x-2) = 0$
- Solutions: $(1,2)$ and $(2,3)$

#### Vieta's Formulas (Sum/Product of Roots)
For $ax^2 + bx + c = 0$ with roots $r_1, r_2$:
- **Sum**: $r_1 + r_2 = -\frac{b}{a}$
- **Product**: $r_1 r_2 = \frac{c}{a}$

**Application**: Find $r_1^2 + r_2^2$ without solving
- $(r_1 + r_2)^2 = r_1^2 + r_2^2 + 2r_1 r_2$
- $r_1^2 + r_2^2 = (-\frac{b}{a})^2 - 2(\frac{c}{a})$

#### Discriminant Analysis
$\Delta = b^2 - 4ac$
- $\Delta > 0$: Two distinct real roots
- $\Delta = 0$: One real root (double/tangent)
- $\Delta < 0$: No real roots (complex)
- **Perfect square** → rational roots → factorable over integers

#### Practice Problems (Math_H2 style)
1. $2x^2 - 7x + 3 = 0$. Sum of roots? Product?
   - Sum = $7/2$, Product = $3/2$

2. $y = -2(x-3)^2 + 5$. Vertex? Max or min? y-intercept?
   - Vertex $(3,5)$, opens down → **max** = 5
   - y-int: $x=0 \rightarrow y = -2(9)+5 = -13$

3. System: $y = 2x+1$ and $y = x^2 - 4x + 7$. How many solutions?
   - $2x+1 = x^2 - 4x + 7 \rightarrow x^2 - 6x + 6 = 0$
   - $\Delta = 36 - 24 = 12 > 0$ → **2 solutions**

4. Find $k$ so $x^2 + kx + 9 = 0$ has exactly one solution.
   - $\Delta = 0 \rightarrow k^2 - 36 = 0 \rightarrow k = \pm 6$
