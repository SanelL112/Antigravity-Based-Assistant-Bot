## 🎯 CORE TRIGONOMETRY: SOH-CAH-TOA

### The Three Primary Ratios
| Ratio | Formula | Memory Aid |
|-------|---------|------------|
| **Sine (sin)** | Opposite / Hypotenuse | **SOH** |
| **Cosine (cos)** | Adjacent / Hypotenuse | **CAH** |
| **Tangent (tan)** | Opposite / Adjacent | **TOA** |

> ⚠️ **Critical:** These ratios are for **acute angles only** (0° < θ < 90°) in a **right triangle**.

### Reciprocal Ratios (Tested Occasionally)
| Ratio | Formula | Relationship |
|-------|---------|--------------|
| **Cosecant (csc)** | Hypotenuse / Opposite | csc θ = 1 / sin θ |
| **Secant (sec)** | Hypotenuse / Adjacent | sec θ = 1 / cos θ |
| **Cotangent (cot)** | Adjacent / Opposite | cot θ = 1 / tan θ |

## 🔄 COMPLEMENTARY ANGLE IDENTITY (HUGE ON SAT)

**For any acute angles A and B where A + B = 90°:**

| Identity | Example |
|----------|---------|
| **sin A = cos B** | sin 30° = cos 60° = 1/2 |
| **cos A = sin B** | cos 30° = sin 60° = √3/2 |
| **tan A = cot B** | tan 30° = cot 60° = √3/3 |

> 🎯 **SAT Favorite:** "If sin θ = 3/5, what is cos(90° - θ)?" → **3/5** (same value!)
> **Also:** "sin θ = cos(90° - θ)" — if sin θ = 0.6, cos(90°-θ) = 0.6

## 📐 SPECIAL RIGHT TRIANGLES (Side Ratios)

### 30°-60°-90° Triangle
```
        /|
       / |
      /  | x√3  (opposite 60°)
     /   |
    /____|
  x      (opposite 30°)
  
Hypotenuse = 2x
Short leg (opp 30°) = x
Long leg (opp 60°) = x√3
```

**Given any one side, find the others:**
- If short leg = 5 → hypotenuse = 10, long leg = 5√3
- If hypotenuse = 14 → short leg = 7, long leg = 7√3
- If long leg = 12√3 → short leg = 12, hypotenuse = 24

### 45°-45°-90° Triangle
```
        /|
       / |
      /  | x
     /   |
    /____|
   x     x
   
Hypotenuse = x√2
Legs = x, x
```

**Given any one side, find the others:**
- If leg = 7 → hypotenuse = 7√2
- If hypotenuse = 10√2 → leg = 10
- If hypotenuse = 12 → leg = 12/√2 = 6√2 (rationalize!)

## 🧮 SOLVING RIGHT TRIANGLES: COMPLETE PROCESS

### Given: One angle + one side
**Example:** Right triangle, ∠A = 30°, hypotenuse = 10. Find other sides.

**Method 1: Special Triangle Ratios (FASTEST)**
- 30° → short leg = hypotenuse/2 = 5
- Long leg = short leg × √3 = 5√3

**Method 2: Trig Ratios**
- sin 30° = opposite/hypotenuse → 1/2 = a/10 → a = 5
- cos 30° = adjacent/hypotenuse → √3/2 = b/10 → b = 5√3

### Given: Two sides
**Example:** Legs = 5 and 12. Find angles.

**Step 1:** Find hypotenuse using Pythagorean Theorem
- c = √(5² + 12²) = √169 = 13

**Step 2:** Find angles using inverse trig
- sin A = 5/13 → A = sin⁻¹(5/13) ≈ 22.6°
- B = 90° - A ≈ 67.4°

> 💡 **SAT usually uses Pythagorean triples:** 3-4-5, 5-12-13, 8-15-17, 7-24-25

## 📐 PYTHAGOREAN THEOREM & CONVERSE

### Pythagorean Theorem
**a² + b² = c²** (c = hypotenuse, a,b = legs)

### Common Pythagorean Triples (MEMORIZE)
| Triple | Multiples |
|--------|-----------|
| 3-4-5 | 6-8-10, 9-12-15, 30-40-50 |
| 5-12-13 | 10-24-26, 15-36-39 |
| 8-15-17 | 16-30-34 |
| 7-24-25 | 14-48-50 |
| 9-40-41 | — |

### Converse of Pythagorean Theorem
| If a² + b² = c² | Triangle is **RIGHT** |
| If a² + b² > c² | Triangle is **ACUTE** |
| If a² + b² < c² | Triangle is **OBTUSE** |

## 🎯 TRIGONOMETRY ON THE COORDINATE PLANE

### Unit Circle Definitions (Extension)
For angle θ in standard position (vertex at origin, initial side on +x-axis):
- **cos θ = x-coordinate** of point on unit circle
- **sin θ = y-coordinate** of point on unit circle
- **tan θ = y/x** (slope of terminal ray)

### Quadrant Signs (ASTC - All Students Take Calculus)
| Quadrant | Angles | sin | cos | tan |
|----------|--------|-----|-----|-----|
| **I** | 0°-90° | + | + | + |
| **II** | 90°-180° | + | - | - |
| **III** | 180°-270° | - | - | + |
| **IV** | 270°-360° | - | + | - |

> ⚠️ **SAT only tests acute angles (0°-90°) in right triangles.** Quadrant signs are for ACT/advanced.

### Coordinate Geometry with Right Triangles
| Formula | Use |
|---------|-----|
| **Distance:** √[(x₂-x₁)² + (y₂-y₁)²] | Side lengths from coordinates |
| **Slope:** (y₂-y₁)/(x₂-x₁) | Determine perpendicular (m₁×m₂ = -1) |
| **Midpoint:** ((x₁+x₂)/2, (y₁+y₂)/2) | Center of hypotenuse = circle center |

**Area on Coordinate Plane (Shoelace):**
```
Area = ½ |x₁(y₂-y₃) + x₂(y₃-y₁) + x₃(y₁-y₂)|
```

### Set A: Basic Trig Ratios
1. **Right triangle, ∠A = 30°, hypotenuse = 20. Find opposite side.**  
   *sin 30° = opp/20 → 1/2 = opp/20 → opp = 10*

2. **Right triangle, ∠B = 60°, adjacent = 8. Find hypotenuse.**  
   *cos 60° = 8/hyp → 1/2 = 8/hyp → hyp = 16*

3. **Right triangle, legs = 5, 12. Find tan of smaller acute angle.**  
   *Smaller angle opposite shorter leg: tan = 5/12*

4. **If sin θ = 3/5 and θ is acute, find cos θ and tan θ.**  
   *3-4-5 triangle: cos = 4/5, tan = 3/4*

5. **sin 45° = ? cos 45° = ? tan 45° = ?**  
   *√2/2, √2/2, 1*

### Set B: Complementary Angle Identity
6. **If sin 25° = 0.423, what is cos 65°?**  
   *0.423 (complementary: 25° + 65° = 90°)*

7. **If cos θ = 12/13, what is sin(90° - θ)?**  
   *12/13 (complementary identity)*

8. **tan 30° = cot ___?**  
   *60°*

9. **sin(90° - x) = 0.8. What is cos x?**  
   *0.8*

### Set C: Special Right Triangles
10. **30-60-90 triangle, short leg = 7. Find other sides.**  
    *Hypotenuse = 14, Long leg = 7√3*

11. **45-45-90 triangle, hypotenuse = 10√2. Find leg.**  
    *10*

12. **45-45-90 triangle, hypotenuse = 12. Find leg.**  
    *12/√2 = 6√2*

13. **30-60-90 triangle, long leg = 10√3. Find hypotenuse.**  
    *Short leg = 10, Hypotenuse = 20*

14. **Right triangle, legs = 9, 12. Find hypotenuse and both acute angles.**  
    *c = √(81+144) = 15 (3-4-5 triple ×3)*

    *B = 90° - 36.9° = 53.1°*

15. **Right triangle, leg = 8, hypotenuse = 17. Find other leg and angles.**  
    *Other leg = √(289-64) = 15 (8-15-17 triple)*  
    *sin A = 8/17 → A ≈ 28.1°*

    *c = √(400+441) = √841 = 29 (20-21-29 triple)*

### Set E: Coordinate Geometry
17. **Right triangle vertices: A(0,0), B(6,0), C(0,8). Find area, hypotenuse, sin/cos/tan of ∠A.**  
    *Legs: 6 and 8 → Area = ½×6×8 = 24*  
    *Hypotenuse = 10 (3-4-5 ×2)*  
    *∠A at origin: opposite = 8, adjacent = 6, hyp = 10*  
    *sin A = 8/10 = 4/5, cos A = 6/10 = 3/5, tan A = 8/6 = 4/3*

18. **Points A(1,2), B(4,6), C(7,2). Is ΔABC right? If so, where is right angle?**  
    *AB = √(3²+4²) = 5*  
    *BC = √(3²+(-4)²) = 5*  
    *AC = 6*  
    *5² + 5² = 50 ≠ 36 = 6² → NOT right*

19. **Right triangle with right angle at origin, legs along axes, hypotenuse endpoints (0,8) and (6,0). Find circle equation through all three vertices.**  
    *Hypotenuse = diameter = 10*  
    *Midpoint of hypotenuse = center = (3,4)*  
    *Radius = 5*  
    *Equation: (x-3)² + (y-4)² = 25*

## 🎯 SAT/ACT-SPECIFIC STRATEGIES

### 1. **Recognize Pythagorean Triples Instantly**
- 3-4-5, 5-12-13, 8-15-17, 7-24-25
- Multiples: 6-8-10, 9-12-15, 10-24-26, etc.
- Saves huge time vs. calculating √(a²+b²)

### 2. **Use Special Triangles Before Trig**
- 30-60-90 or 45-45-90 → Use side ratios
- Much faster than sin/cos/tan + inverse

### 3. **Complementary Identity = Free Points**
- sin θ = cos(90°-θ) — if you see this, answer is IMMEDIATE
- sin 35° = cos 55°, sin 17° = cos 73°, etc.

### 4. **Rationalize Denominators**
- √2/2 not 1/√2
- √3/3 not 1/√3
- SAT answer choices use rationalized form

### 5. **Draw the Triangle!**
- Label angle, opposite, adjacent, hypotenuse
- Visual prevents "which side is which" errors

### 6. **Coordinate Plane Right Triangles**
- Legs parallel to axes → easy lengths (difference in x or y)
- Hypotenuse = distance formula
- Midpoint of hypotenuse = center of circumscribed circle

### 7. **Desmos Verification**
- Plot triangle points
- Measure distances/angles
- Graph circle through vertices

## ❌ COMMON TRAPS TO AVOID

| Trap | Wrong | Correct |
|------|-------|---------|
| **Opposite vs Adjacent** | Mixing up sides for angle | Label triangle: Opposite = across from angle |
| **Hypotenuse** | Using leg as hypotenuse | Hypotenuse is ALWAYS longest side, opposite 90° |
| **Rationalizing** | 1/√2, 1/√3 | √2/2, √3/3 |
| **Angle Mode** | Calculator in radians | Ensure DEGREES mode |
| **Inverse Trig** | sin⁻¹(0.5) = 30° but calculator gives radians | Use DEGREES mode |
| **Complementary** | sin θ = sin(90°-θ) | sin θ = cos(90°-θ) |
| **45-45-90 Hypotenuse** | Leg × 2 | Leg × √2 |
| **30-60-90 Long Leg** | Short × 2 | Short × √3 |
